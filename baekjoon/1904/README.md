백준 [01타일](https://www.acmicpc.net/problem/1904)(1904번)

# 문제 해석 🤔
1. 00과 1을 이용해 길이가 N인 모든 경우의 수를 구하는 문제.
2. 규칙을 찾기 위해 계산한 결과 
N=1 -> 1개
N=2 -> 2개
N=3 -> 3개
N=4 -> 5개
N=5 -> 8개
과 같이 구성된다.
3. 피보나치 수열과 동일한 규칙성을 가짐!!
